# 💈 Telegram Barber Bot

A Telegram bot for a barber shop with online booking functionality and Google Calendar integration.

---

## 🚀 Features

- 📅 Online appointment booking
- 🤖 User-friendly Telegram interface
- 🗓 Google Calendar integration
- ⏰ Schedule management

---

## 🛠 Tech Stack

- Python 3
- aiogram
- Google Calendar API

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/N0va-El/telegram-barber-bot.git
cd telegram-barber-bot
pip install -r requirements.txt
python bot.py
```
## 🔐 Security

Do NOT upload the following files:
- `.env`
- `client_secret.json`
- `token.json`
